$* 和 $@ 有什么区别？

args=("$@") 和 args=("$@")的区别？ 

<hr>

- `$*` ，所有的命令行参数被看作是一个单一的字符串, 空格分割
- `$@` 是一个数组，每个参数都是数组的一个元素


`args=("$@")`
`args=($@)`

使用 "$@" 时，每个参数都被双引号包围。这意味着每个参数都被视为一个独立的元素，即使它包含空格。这种方法可以正确处理包含空格的参数

使用 `$@` 时，参数中的空格将作为分隔符，导致包含空格的参数被拆分为多个数组元素

```shell
args=("$*")
echo "${args[0]}"

args=("$@")
echo "${args[0]}"
```
